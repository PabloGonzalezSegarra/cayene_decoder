# Cayene Decoder

A modern C++26 library for decoding Cayene LPP (Low Power Payload) format.

## Requirements

- **Compiler**: Clang 18+ (with C++26 support)
- **CMake**: 3.25+
- **Dependencies**: Automatically fetched via FetchContent

## Building

```bash
# Configure with Clang
cmake -B build -S . \
    -DCMAKE_CXX_COMPILER=clang++ \
    -DCMAKE_BUILD_TYPE=Release

# Build
cmake --build build

# Run tests
ctest --test-dir build --output-on-failure

# Run example
./build/examples/basic_example
```

### Build Options

| Option | Default | Description |
|--------|---------|-------------|
| `CAYENE_BUILD_TESTS` | ON | Build unit tests |
| `CAYENE_BUILD_EXAMPLES` | ON | Build examples |
| `CAYENE_ENABLE_WARNINGS` | ON | Enable compiler warnings |
| `CAYENE_ENABLE_SANITIZERS` | OFF | Enable ASan/UBSan in Debug |

### Debug Build with Sanitizers

```bash
cmake -B build -S . \
    -DCMAKE_CXX_COMPILER=clang++ \
    -DCMAKE_BUILD_TYPE=Debug \
    -DCAYENE_ENABLE_SANITIZERS=ON

cmake --build build
```

## Adding Dependencies

To add new dependencies, use CMake's FetchContent in `CMakeLists.txt`:

```cmake
FetchContent_Declare(
    new_dependency
    GIT_REPOSITORY https://github.com/owner/repo.git
    GIT_TAG v1.0.0
)
FetchContent_MakeAvailable(new_dependency)
```
